# 🌐 ConnectHub – Full Stack Social Media Platform

ConnectHub is a modern Full Stack Social Media Web Application where users can connect, share posts, follow other users, like and comment on posts, explore trending creators, and chat with each other.

---

## 🚀 Features

### 👤 Authentication
- User Registration
- User Login
- JWT Authentication
- Secure Password Hashing

### 📝 Posts
- Create Post
- Upload Images
- Delete Posts
- Feed Page

### ❤️ Social Features
- Like Posts
- Comment on Posts
- Follow / Unfollow Users
- Explore Users
- Trending Users

### 👤 User Profile
- View Profile
- Edit Profile
- Update Profile Picture
- User Search

### 💬 Messaging
- One-to-One Chat
- Real-time Style Auto Refresh
- Unread Message Badge

### 🎨 UI
- Responsive Design
- Modern User Interface
- Clean Layout
- Mobile Friendly
